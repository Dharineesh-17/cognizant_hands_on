1. DatePipe — date

Format Date objects and ISO strings using predefined or custom format strings.

{{ '{{ now | date: "fullDate" }}' }} {{ now | date:'fullDate' }}
{{ '{{ now | date: "shortDate" }}' }} {{ now | date:'shortDate' }}
{{ '{{ now | date: "medium" }}' }} {{ now | date:'medium' }}
{{ '{{ now | date: "yMMMd" }}' }} {{ now | date:'yMMMd' }}
{{ '{{ now | date: "HH:mm:ss" }}' }} {{ now | date:'HH:mm:ss' }}
{{ '{{ now | date: "MMMM yyyy" }}' }} {{ now | date:'MMMM yyyy' }}
Formats: shortDate | mediumDate | longDate | fullDate | shortTime | HH:mm:ss | MMMM dd, yyyy

2. String Pipes — uppercase, lowercase, titlecase

Transform string casing in the template.

Original: {{ studentName }}
uppercase {{ studentName | uppercase }}
titlecase {{ studentName | titlecase }}
Category: {{ courseCategory }}
lowercase {{ courseCategory | lowercase }}
titlecase {{ courseCategory | titlecase }}

3. SlicePipe — slice:start:end

Extract a subset of an array or string (similar to Array.slice()).

String slice:

{{ longDescription }}

{{ longDescription | slice:sliceStart:sliceEnd }}...

Array slice (top students):

{{ student }}

Showing {{ sliceEnd - sliceStart }} of {{ topStudents.length }} students

{{ "{{ tags | slice:0:3 }}" }} → first 3 elements | {{ "{{ text | slice:0:50 }}" }} → truncate text

4. Currency, Percent & Number Pipes

Course Price: {{ coursePrice | currency }}
currency:"EUR":"symbol":"1.2-2" {{ coursePrice | currency:'EUR':'symbol':'1.2-2' }}
{{ '{{ annualSalary | currency }}' }} {{ annualSalary | currency }}
Completion Rate: {{ completionRate | percent }}
percent:"1.2-2" {{ completionRate | percent:'1.2-2' }}
GPA: {{ gpa | number:'1.2-2' }}
number:"2.1-1" {{ gpa | number:'2.1-1' }}
Total Enrolled: {{ totalEnrolled | number }}
number:'1.2-2' = min 1 digit before, min 2-max 2 after decimal | currency:'EUR' = Euro symbol

5. JsonPipe — json

Debug: convert any value to pretty-printed JSON string.

{{ courseObject | json }}
Use {{ '{{ data | json }}' }} for debugging — formats any object as pretty JSON in the template.

6. AsyncPipe — async

Auto-subscribe to Observables/Promises. Auto-unsubscribes on destroy.

Observable Message (2s delay)

{{ message }}

Loading...

Live Clock (every second)

{{ clock$ | async | date:'HH:mm:ss' }}

Student Data (1.5s delay)

{{ data.name }}

Course: {{ data.course }}

Progress: {{ data.progress }}%

Fetching student data...
{{ '{{ observable$ | async }}' }} — auto-subscribes & auto-unsubscribes on destroy. No manual cleanup!

7. I18n Plural & Select Pipes

I18nPluralPipe

{{ enrolledCount }}

{{ enrolledCount | i18nPlural: { '=0': 'No students enrolled', '=1': 'One student enrolled', 'other': '# students enrolled' } }}

I18nSelectPipe

{{ studentGender | i18nSelect: genderMap }} Alice Williams

Status: {{ enrollmentStatus | i18nSelect: statusMap }}

i18nPlural selects by '=0', '=1', or 'other' | i18nSelect selects by any key
